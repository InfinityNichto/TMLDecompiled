static Dictionary< int, Dictionary< int, int > > SpecificLauncherAmmoProjectileMatches
 Associates a launcher's item type (F:Terraria.Item.type) and an ammo's item type (F:Terraria.Item.type) to the projectile type (F:Terraria.Projectile.type) they will shoot when used together.
