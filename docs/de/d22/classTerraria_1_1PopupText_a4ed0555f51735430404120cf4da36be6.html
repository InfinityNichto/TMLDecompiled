Creates a new T:Terraria.PopupText in F:Terraria.Main.popupText at the center of the picked-up newItem .
If a T:Terraria.PopupText already exists with the M:Terraria.Item.AffixName of newItem , that text will instead be modified unless noStack is true.

Parameters
contextThe T:Terraria.PopupTextContext in which this T:Terraria.PopupText was created.
newItemThe T:Terraria.Item to ceate the new text from.
stackThe stack of newItem .
noStackIf true, always create a new T:Terraria.PopupText instead of modifying an existing one.
longTextIf true, then this T:Terraria.PopupText will spawn with F:Terraria.PopupText.lifeTime == 5 * 60.
Returns

Also returns -1 if P:Terraria.Item.Name is null.
