◆ noGravity

bool Terraria.Dust.noGravity

Indicates if a dust should be affected by gravity or not. Not all vanilla dust have logic checking this value and modded dust with custom M:Terraria.ModLoader.ModDust.Update(Terraria.Dust) code would need to implement this into the logic for it to have effect. Defaults to false.
