void Terraria.ModLoader.Core.TmodFile.AddFile ( string fileName,

inlinepackage

Adds a (fileName -> content) entry to the compressed payload This method is not threadsafe with reads, but is threadsafe with multiple concurrent AddFile calls.

Parameters
fileNameThe internal filepath, will be slash sanitised automatically
dataThe file content to add. WARNING, data is kept as a shallow copy, so modifications to the passed byte array will affect file content
