bool Terraria.IO.WorldFileData.TryGetModVersionGeneratedWith ( string mod,
out Version modVersion )
inline

Retrieves the version that the specified mod was at when the world was generated. false will be returned for mods that were not enabled when the world was generated.

The feature tracking which mods were used to generate a world was added in v2023.8, so modders should first check P:Terraria.IO.WorldFileData.WorldGenModsRecorded to see if the mods used to generate the world were recorded at all.
