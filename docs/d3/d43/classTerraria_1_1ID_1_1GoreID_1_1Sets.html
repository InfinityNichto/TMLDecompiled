 If non-zero for a given gore type (F:Terraria.Gore.type), that gore will follow special AI rules.
For 1, this gore will act like a piece of sail, such as the ones dropped by a Flying Dutchman.
For 2, this gore will act like chimney smoke. This is used by the smoke particles created by the F:Terraria.ID.TileID.Chimney.
For 3, this gore will act like a falling leaf. This is used for all vanilla tree leaves.
For 4, this gore will float along the ground. This is used by grounded graveyard fog.
For 5, this gore will float in the air. This is used by airborne graveyard fog.
For 6, this gore will slowly diffuse into open air. This is used by the fog created by the F:Terraria.ID.TileID.FogMachine.
For 7, this gore will quickly fade out. This is used by the spark particles created by the F:Terraria.ID.ProjectileID.VoltBunny.
Defaults to 0.
